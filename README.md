# Literacy Rate Impact Analysis 📊

## 📌 Project Overview
This project analyzes the impact of literacy rate on:
- GDP
- Unemployment Rate

The goal is to understand how education levels influence economic growth and employment.

---

## 📂 Project Structure
```
Literacy_Rate_Impact_Analysis/
│
├── literacy_rate_impact_analysis.py
├── literacy_rate_impact_analysis.ipynb
├── literacy_data.csv
├── requirements.txt
└── README.md
```

---

## 📊 Features
- Data Cleaning
- Missing Value Detection
- Duplicate Removal
- Negative Value Checking
- Correlation Analysis
- Data Visualization using Matplotlib

---

## 🚀 How to Run

### Install Requirements
```
pip install -r requirements.txt
```

### Run Python Script
```
python literacy_rate_impact_analysis.py
```

- Literacy Rate vs Unemployment Rate
- Correlation Matrix Heatmap

## 🎯 Conclusion
Higher literacy rates generally show positive economic impact and lower unemployment trends (based on sample data).
